# Java Rectangle OOP Implementation

This project is a fundamental exercise in **Object-Oriented Programming (OOP)** using Java. It features a robust `Rectangle` class and a driver program to simulate object interaction.

## 🚀 Features
* **Encapsulation:** Uses private instance variables with public accessors (getters) and mutators (setters).
* **Multiple Constructors:** Demonstrates constructor overloading to initialize objects with varying levels of detail.
* **Logic Implementation:** Includes methods for calculating geometric properties like Area and Perimeter.
* **Driver Program:** A test-driven approach to instantiate and manipulate multiple objects.

## 🛠️ Concepts Applied
- **Classes & Objects**
- **Data Hiding (Encapsulation)**
- **Method Implementation**
- **Overloading**
